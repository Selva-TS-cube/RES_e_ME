Project Name: RESeME
Team Name : Unique
Creators: Selvendran S. , Thadeus Cruz Govindapillai

This project mainly focuses on resume building for all users including students who
like creating resume for encouragement. We hope that our project works as expected.

This project includes Login/signup page, A page to enter the necessary details of the user,
 and lastly we have the resume building page where one can get the abstract of all the 
achievements and education background.

We have an additional feature of taking a quiz on various topics such as HTML, CSS, 
Machine Learning and so on.. Currently we have added only HTML Quiz. The purpose of this 
feature is to grade the user and based on the results of the quiz, the lines of the final 
output page will be enhanced.
This feature allows every user to take the quiz and add the resume lines which are not included
in the uploaded user entry data.
